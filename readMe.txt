【プロジェクトの概要】
    このプロジェクトはlabelPrinterへのオーダー印刷デモです。
    このプロジェクトはコンテンツを全てローカル実行できるように作成しています。
    そのためブラウザとプリンタをｌａｎ接続すれだけで、webServer無しで印刷できます。

【セットアップ】
    プロジェクトを任意の場所に解凍し、解凍フォルダから起動.batを実行ください。
    起動後、設定ボタンを押し、プリンタのIPアドレスを設定ください。

【使い方】
    オーダーボタンを押すとオーダーがエントリされ、印刷を試みます。
    印刷が成功すると、オーダー情報の背景地が黄色くなります。
    オーダー情報をタップすると、オーダーを消すことができます。

【使用ライブラリ】
    jQuery 3.6.0


 【開発履歴】   
    送信方法について										
        ePOS-SDK非対応の機種のため、XHRを使って80番ポートに送信する。									
        XHRのひな型は？									
                                            
    xhrクラスを作り、独自のコールバックの仕組みを作ってみる。（但し最初は単純なsettimeoutするだけでいい）										
                                            
    各クラス(VC,MC,DCおよびxhr)の役割分担が決まり、										
    vc:画面操作　mc:データ生成およびクラス間連携指示　dc:印刷制御　xhr:データ送受信										
    アプリの骨格を作れた。またプリンタとの印刷およびステータス確認の挙動を実現した。										
                                            
    次は表示に必要な情報の追加、表示内容の追加および修正と、それに対応するDC,MCからの情報提供の流れを作る。										
                                            
    次に画面周りに手入れし、想定するオペレーションを実現させる。										
                                            
    次に各OS環境で動作できるよう設定まわりとドキュメントを作成する。
